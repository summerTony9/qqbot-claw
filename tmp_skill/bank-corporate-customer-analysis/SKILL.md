# 对公客户深挖分析

## 使用方式

用户给公司名称 → 按下述步骤执行 → 输出标准分析报告。

## 执行步骤

### 第一步：读取分析模板
读取 `references/analysis-template.md`，获取完整分析框架和评分标准。

### 第二步：搜索企业基础信息
使用 agent-browser 打开百度搜索：
```
北京+[公司名称]+公司
```
重点抓取：
- 爱企查/天眼查的企业信息卡片（工商信息、注册资本、法人、成立时间）
- 企业性质（民营/国企/上市公司/外资）
- 高新技术企业、专精特新认定
- 主营业务描述
- 企业地址和联系方式

### 第三步：搜索业务和行业信息
搜索：
```
北京+[公司名称]+招投标
北京+[公司名称]+主营+行业
```
重点抓取：
- 近期招投标记录（近6-12个月）
- 主营业务和行业赛道
- 核心产品/技术
- 是否属于热点方向（AI/新能源/半导体/高端制造等）

### 第四步：搜索股东和集团关系
搜索：
```
北京+[公司名称]+对外投资+子公司
北京+[公司名称]+母公司+股东
```
重点抓取：
- 投资方数量
- 对外投资数量
- 疑似控股数量
- 分支机构数量
- 是否有集团背景

### 第四步（补充）：深度核查母子公司辐射关系【必须执行】

在第四步抓取到"对外投资数量"和"疑似控股数量"后，必须进一步判断：

**1. 自身对外辐射（子公司情况）**
- 对外投资 N 家 → 记录这 N 家公司的名称和行业
- 每家子公司注册资本多少，是否独立经营
- 是否有子公司在其他银行已有深度合作

**2. 实控人关联企业网络（自然人股东名下企业）**
- 查实控人（自然人）是否控股或担任多家企业高管
- 这些关联企业加起来体量有多大
- 是否形成"集团化经营"特征

**3. 是否有母公司（上属关系）**
- 该公司本身是否是某个大公司的子公司
- 母公司是谁，体量和行业如何
- 母公司在不在你的客户名单里

**4. 辐射价值判断（最重要）**
| 情形 | 辐射价值 | 评级修正 |
|------|---------|---------|
| 体量小但母公司体量大 | 以母公司为切入点，子公司自然延伸 | C→B 或 B→A |
| 体量小但有多个子公司 | 通过该公司触达多家子公司做集团联动 | C→B |
| 体量小，母公司和子公司体量都小 | 无辐射价值，维持原评级 | 不变 |
| 体量大且有子公司群 | 集团整体授信机会，各子公司分别开发 | 维持A/B |

**核心逻辑：** 体量小的公司不一定是"价值低"——如果它是某个大集团的子公司，或者一个自然人控制多家企业，那这个实控人/母公司就是你的杠杆，撬动的是一个企业群。

### 第五步：搜索近期动态
搜索：
```
北京+[公司名称]+招聘
北京+[公司名称]+专利+融资+中标
```
重点抓取：
- 招聘动态（判断扩张/收缩）
- 专利申请（技术创新信号）
- 融资事件（是否有资本进入）
- 中标记录（大项目信号）

### 第六步：如有官网，访问官网
搜索公司官网并访问，获取：
- 主营业务详细描述
- 产品信息
- 合作伙伴
- 联系方式
- 最新动态

### 第七步：输出分析报告
按 `references/analysis-template.md` 中的模板格式输出，包含：
1. 客户基础信息
2. 核心结论摘要（3句话）
3. 企业经营与行业判断
4. 股东背景与集团关系
   - **★★★ 特别重要：母子公司辐射分析（必须填写）**
     - 该公司对外投资的子公司有哪些，辐射价值如何
     - 实控人名下是否有关联企业网络
     - 是否有母公司，母公司辐射价值如何
     - 评级是否需要因辐射价值修正
5. 上下游与商业生态分析
6. 近期动态与增长信号
7. 资金流与银行业务场景分析
8. 风险与障碍判断
9. 机会分类判断
10. 营销切入建议
11. 综合评分（A/B/C/D类）——**母子公司辐射价值作为单独评分项**
12. 6行总结——**第3行专门写母子公司辐射机会**

## 评级标准

- **A类**：重点深挖，优先拜访。行业热点+扩张信号强+金融机会明确
- **B类**：持续跟踪，择机切入。有真实业务机会但时机未到或需进一步确认
- **C类**：维护观察，以结算为主。体量有限但有合作价值，以存款/结算为主
- **D类**：暂不重点投入。风险大于机会或他行深度绑定

## 普惠金融注意事项

即使体量小的客户：
- 高新技术企业 ✅ → 科创贷适用
- 有稳定大客户TO-B ✅ → 供应链融资适用
- 有招投标记录 ✅ → 保函适用
- 有真实经营 ✅ → 代发/结算适用
- 有专利技术 ✅ → 轻资产评估融资适用

普惠不看规模，看：真实经营 + 稳定客户 + 技术/资产支撑
