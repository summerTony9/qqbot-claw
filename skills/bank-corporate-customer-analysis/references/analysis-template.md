> 更新：2026-03-26

## 使用说明

本模板用于对公客户批量深挖分析，按以下顺序执行：

1. 搜索企业基础工商信息
2. 搜索主营业务和行业赛道
3. 搜索股东背景和集团关系
4. 搜索对外投资和子公司
5. 搜索上下游客户情况
6. 搜索近期重大动态（招投标/融资/招聘）
7. 搜索官网和新闻
8. 按本模板输出分析结果

---

## ⚠️ 第一步：判断企业性质（先定类型，再定分析逻辑）

**在填任何分析之前，必须先判断这个企业是哪种类型：**

| 企业性质 | 特征 | 核心分析逻辑 |
|---------|------|------------|
| **运营实体** | 有实际业务、员工、产品 | 按正常经营逻辑分析，参考员工规模 |
| **控股公司/母公司** | 无实际业务或业务极少、员工为0或很少、注册资本低、持有其他公司股权 | **不能看自身营收，要看持有的资产价值** |
| **混合型** | 部分自营业务+部分股权投资 | 拆分两部分分别分析 |

**⚠️ 特别注意：以下情况不要作为负面信号：**
- 控股公司/母公司的员工人数为0或很少 → **正常现象，不扣分**
- 控股公司/母公司的注册资本比子公司小 → **正常现象，不扣分**
- 控股公司/母公司经营范围窄/主要是投资类 → **正常现象，要看持有的子公司**

**判断依据：**
- [ ] 是否有大量对外投资（持股子公司）？
- [ ] 是否基本无实际业务运营？
- [ ] 是否为XX集团/XX投资的母公司？
- [ ] 员工人数是否为0或极低？
- [ ] 营业执照经营范围是否主要是"投资""企业管理""咨询"？

**满足3条及以上 → 判定为控股公司/母公司，按控股公司逻辑分析。**

## 五、上下游与商业生态分析

### 1. 下游客户情况
- 主要客户是谁
- 是否服务大企业、央国企、上市公司、知名品牌
- 客户集中度是否高
- 下游客户是否稳定

### 2. 上游供应商情况
- 核心供应商是谁
- 原材料/核心资源是否受制于人
- 上游议价能力强不强

### 3. 产业链判断
- 在链条里处于什么位置
- 是核心节点还是普通配套
- 是否进入重点产业链、龙头企业供应链

### 4. 对银行的意义
- 是否可挖供应链金融
- 是否可延伸开发上下游客户
- 是否存在应收/预付款融资场景

## 八、风险与障碍判断

### 1. 经营风险
- 行业是否下行
- 是否过度依赖单一客户
- 是否毛利低、竞争激烈
- 是否扩张过快

### 2. 法律与信用风险
- 是否有被执行、失信、涉诉
- 是否有税务异常、环保处罚
- 是否有股权冻结、频繁工商变更

### 3. 银行切入障碍
- 是否已被他行深度绑定
- 是否偏股权融资、不偏银行融资
- 是否轻资产、缺少抵质押
- 是否决策链条长、触达困难

### 4. 风险结论
- **可重点营销**
- **谨慎营销**
- **先维护观察**
- **暂不建议投入**
