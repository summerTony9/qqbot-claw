# 小红书北京企业融资监控

第一版目标：每小时巡检一次“小红书中与北京企业贷款/融资相关的帖子”，抽取线索并去重。

## 当前实现状态

- 已配置关键词池
- 已实现本地状态去重
- 已实现 relay 健康检查与采集骨架
- 已预留输出字段：公司名、用户ID、昵称、内容、链接、时间、命中词、线索等级
- 已预留 cron 调度

## 运行前提

第一版采集依赖 `browser-relay-xiaohongshu`：

1. 本机/桌面环境有已登录的小红书 Chromium
2. 已开启 CDP 9222
3. relay 已启动并可通过 `127.0.0.1:18792` 访问
4. `/tmp/browser-relay-token` 可读

如果 relay 不可用，脚本会输出 `relay_unavailable`，不会误报。
